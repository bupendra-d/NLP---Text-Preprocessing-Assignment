## Text Preprocessing using Python (NLP)

# Overview
 - This assignment demonstrates basic text preprocessing techniques used in Natural Language Processing (NLP). Two paragraphs are used as input and processed to clean the text before analysis.

# Techniques Used
- Punctuation Removal – Removes punctuation marks such as . , : ; ! ?
- Stop Word Removal – Removes common words like is, the, and, in
- Stemming – Reduces words to their root form using Porter Stemmer
- Lemmatization – Converts words to their meaningful base form using WordNet Lemmatizer

# Technologies
- Python
- NLTK
  
# Workflow
- Input Text → Tokenization → Punctuation Removal → Stop Word Removal → Stemming → Lemmatization
